organism moduleÂ¶

Download protein structures for an organism from the RCSB PDB database.

UsageÂ¶

python organism.py --organism_dir organism_dir [--n_jobs n_jobs]

AlgorithmÂ¶

  1. If the ids are already downloaded (a _ids_<date>.txt file with current date exists), skip to the next step.

  2. Start by downloading the RCSB PDB IDs for the organism, using the queries in the queries directory.

  3. Before downloading the PDB files, check which PDB files are already in the local organism directory, and skip those to save time.

  4. Some local PDB files are not in the RCSB database anymore, so we mark them with a suffix (for example â€˜.removedâ€™).

  5. Print a numeric report of: - the number of PDB files already in the organism directory; - the number of PDB files that will be downloaded; - the number of removed/obsolete PDB files.

  6. Download the PDB files corresponding to the RCSB PDB IDs which are not already in the organism directory.

  7. During the download, report the global progress and the expected time to completion.
