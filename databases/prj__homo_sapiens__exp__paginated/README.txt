Progetto RCSB-Sync per scaricare proteine sperimentali di Homo sapiens.

Metodi sperimentali inclusi = X-RAY DIFFRACTION + SOLUTION NMR + ELECTRON MICROSCOPY

Ci sono diverse query, ciascuna delle quali scaricherà 10,000 proteine in una cartella separata.
Questo serve per non appesantire troppo le singole cartelle, comprimerle, condividere file più comodamente, eccetera.

Lancio:

    $ python src/project.py templates/prj__homo_sapiens__exp__paginated -s
